Clone automatically a copy of github repository locally by giving only username and repo name

## Install

```cli
npm install -g gcopy
```

or

```cli
yarn global add gcopy
```
## Using gcopy CLI

#### Command Line

```cli
$ gcopy <options>
```
#### example
```cli
$ gcopy knowbee/gcopy
```

## Options:

- github_username/repo_name
